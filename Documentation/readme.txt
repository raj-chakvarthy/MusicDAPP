MusicDapp
MusicDapp is a Distributed Application (Dapp) running on the Ethereum Blockchain (Rinkeby Network).

***********************************************************************************************************************

Features of MusicDapp:

-Create Concerts and store the details on a Smart Contract
-Concert Images are stored on IPFS 
-View created concerts 
-Allows a user to buy Ticket using metamask wallet provider
-Submission price collected by the contract: 0.001 ETH

***********************************************************************************************************************

Deployed Contract on Rinkeby: 0x0b75A0D40443a77970fa4bb353888F48dA2c6740

***********************************************************************************************************************

Tools

-Smart contracts :    Solidity
-Front End:           Next.JS, React.JS
-Wallet provider:     MetaMask

***********************************************************************************************************************
To Run Locally


- NodeJS version: 8.11.3  is required
- Download and extract MusicDapp folder
- Open MusicDapp folder from command line / terminal
- Run : npm run dev
- Local node server will be established on localhost:3000, 
- Access MusicDapp from localhost:3000
- Refer to Aubout MusicDapp.pdf for detailed user guide
